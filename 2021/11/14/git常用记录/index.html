第五章 任务管理相关

分支

  • 创建分支

git checkout -b dev <==> git switch -c dev

  • 切换分支

git checkout master <==> git switch master

  • 删除分支

git branch -d dev

标签

  • 本地打标签

    git tag -a v1.0 -m 说明

  • 本地打完标签推远端

    git push origin v1.0 OR

    git push origin --tags

  • 本地删标签

    git tag -d v1.0

  • 本地删标签后推远端 :ref/tags/标签名

    git push origin :refs/tags/v1.0
